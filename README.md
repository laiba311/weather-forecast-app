# weather-forecast-app
Flutter weather application with real-time weather tracking, city-based search, OpenWeather API integration, animated user interface, and dynamic weather visualization.

## Features

- Real-Time Weather Updates
- City-Based Weather Search
- OpenWeather API Integration
- Temperature Monitoring
- Humidity Tracking
- Dynamic Weather Themes
- Weather Condition Detection
- Error Handling for Invalid Cities
- Responsive Design
- Modern Animated Interface

## Technologies

- Flutter
- Dart
- OpenWeather API
- HTTP Package
- Material Design

## Key Functionalities

- Search weather by city name
- Fetch live weather data from API
- Display temperature and humidity
- Show weather conditions dynamically
- Handle network and API errors
- Update interface based on weather conditions
